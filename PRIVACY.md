# InfoLens 隐私政策

**最后更新日期**：2026年7月1日

## 1. 我们收集什么

InfoLens **不收集**以下个人信息：
- 您的真实身份、姓名、邮箱
- 您的浏览历史、搜索记录
- 您的 IP 地址、设备指纹
- 您访问的具体网页内容

我们**仅收集**：
- 随机生成的匿名用户 ID（UUID v4，本地生成，不与任何个人身份关联）
- 您主动标注的 URL 和标注类型（如 "👍 值得看"）

## 2. 数据用途

收集的数据仅用于：
- 计算域名可信度评分
- 构建社区共识（≥3 人同一标注类型）
- 在其他用户的搜索结果中显示标注

## 3. 数据存储

- **本地数据**：存储在您的浏览器 `chrome.storage.local` 和 `Service Worker` 内存中
- **云端数据**：仅标注数据（URL + 标签类型）通过 HTTPS 传输至 Cloudflare D1 数据库
- **不存储**：个人身份信息、浏览历史、搜索内容

## 4. 数据共享

- 标注数据作为**社区共识**共享给其他 InfoLens 用户
- 数据以匿名形式存在，无法关联到任何个人
- 不向第三方出售或共享个人数据

## 5. 用户权利

- **查看**：点击扩展图标可查看您的标注历史
- **修改**：对已标注的网页可重新标注或取消
- **导出**：通过弹出面板的"导出"按钮下载您的数据（JSON 格式）
- **删除**：通过弹出面板的"清除所有数据"按钮可删除全部本地数据
- **注销**：清除数据后，您的匿名用户 ID 将被重新生成，历史标注不再关联

## 6. 权限说明

| 权限 | 用途 |
|------|------|
| `<all_urls>` | 在所有网页上显示标注栏，在搜索引擎结果页注入标注徽章 |
| `storage` | 跨页面共享标注数据，持久化用户偏好 |
| `notifications` | 通知用户有新版本可用 |

## 7. 数据安全

- 所有数据传输使用 HTTPS 加密
- 本地数据仅存储在您的设备上
- 云端数据库由 Cloudflare 提供，支持数据加密和访问控制

## 8. 儿童隐私

InfoLens 不面向 13 岁以下儿童。我们不会刻意收集儿童个人信息。

## 9. 政策变更

如隐私政策发生重大变更，我们将通过扩展更新通知用户。
